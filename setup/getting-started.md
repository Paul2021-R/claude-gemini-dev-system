# Getting Started

## Repository Structure

```bash
cloud-agent/
├── CLAUDE.md              # Claude 지침
├── GEMINI.md              # Gemini 지침
├── project-docs/          # 프로젝트 문서 (이 저장소)
└── cloud-agent/           # 실제 코드가 작성될 폴더 (현재 빈 상태)
```

## Setup

1. `project-docs/` 브랜치를 확인한다.
2. `cloud-agent/` 폴더 내에 기술 스택에 맞는 서브 프로젝트를 생성한다.
3. 생성된 서브 프로젝트에 따라 이 문서를 업데이트한다.
