# Owner 분배 기본값

> init에서 사용자가 별도 지정하지 않을 경우 이 기본값을 적용한다.

## 사람 (human) — 직접 작성

- 서비스 핵심 비즈니스 로직
- 보안/인증 설계
- 아키텍처 결정이 필요한 작업
- 외부 서비스 연동 설계
- 데이터 모델링 (스키마 설계)

## 에이전트 (agent) — 위임 가능

- CRUD 엔드포인트, DTO/엔티티 생성
- 테스트 코드 작성
- 설정 파일, 마이그레이션 스크립트
- API 문서화
- 린트/포맷터 설정
- 보일러플레이트, 유틸리티 함수
- CI/CD 파이프라인 스크립트
- 리팩토링 (명확한 지시 하에)
